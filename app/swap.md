# Swap

The Swap tab lets you deposit supported input assets and receive Vetro protocol assets (VUSD or vetBTC) at a 1:1 rate minus a nominal minting fee.

## Supported Swap Inputs (Live)

| You swap | You receive | Status |
|-------------|-------------|--------|
| USDC | VUSD | ✅ Live |
| USDT | VUSD | ✅ Live |
| frxUSD | VUSD | ✅ Live |
| hemiBTC | vetBTC | ✅ Live |
| cbBTC | vetBTC | ✅ Live |
| WBTC | vetBTC | ✅ Live |
